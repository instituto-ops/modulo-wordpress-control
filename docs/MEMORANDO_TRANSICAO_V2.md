# 🧠 Memorando de Transição: Protocolo NeuroEngine v2.0

**Assunto:** Migração de Ecossistema Digital – Do WordPress para Jamstack (Next.js + Vercel)

## 1. Justificativa da Mudança (O Fim do Legado)
O sistema anterior baseado em WordPress operava como um "monolito convidado" em servidores compartilhados (Hostinger), gerando gargalos de performance e limitações de design. A transição para o modelo Jamstack elimina a dependência de bancos de dados lentos e plugins pesados, movendo o processamento para a "borda" (Edge Computing) da Vercel.

## 2. Nova Arquitetura Técnica
O projeto NeuroEngine agora integra os seguintes pilares:

- **Core:** Next.js 15 + TypeScript para uma estrutura de código tipada e segura.
- **Interface:** Tailwind CSS e Framer Motion para animações de nível cinematográfico e design "Dark Premium".
- **Deploy & Hosting:** Vercel (Infraestrutura Global) conectada ao repositório GitHub para Continuous Deployment (CD).
- **Domínio:** [hipnolawrence.com](https://hipnolawrence.com) agora aponta diretamente para os Nameservers da Vercel, garantindo SSL automático e propagação global instantânea.

## 3. Capacidades Implementadas (Célula HipnoLawrence)
O sistema Antigravity consolidou as seguintes rotas e funcionalidades que agora servem de modelo para o NeuroEngine:

- **Roteamento Dinâmico:** Estrutura para artigos editoriais (ex: `/blog/masking-e-exaustao`) que carregam instantaneamente.
- **SEO de Elite:** Implementação de `metadataBase`, Sitemaps automáticos e `Robots.txt` configurados para indexação prioritária no Google.
- **Branding Integrado:** Incorporação do logotipo oficial e tipografia curada (Fraunces & Manrope) diretamente no código, sem dependência de temas externos.

## 4. Integração com NeuroStrategy OS
Diferente do WordPress, a nova integração permite que o NeuroEngine conecte-se diretamente a APIs e bancos de dados (como Supabase ou Prisma) para:

- Criação de áreas de membros para pacientes.
- Dashboards de evolução clínica em tempo real.
- Automação de agendamentos via formulários interativos com validação do lado do servidor.

## ✅ Status de Operação
O domínio oficial [hipnolawrence.com](https://hipnolawrence.com) está **ACTIVE** e operando 100% fora do ambiente WordPress. O NeuroEngine deve agora considerar este repositório (`instituto-ops/hipnolawrence-site`) como a base de produção para futuras expansões de IA e ferramentas de psicologia clínica.
